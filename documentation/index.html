Thank you for purchasing VetAnimative - Premium Pet Clinic HTML Template.

1. Introduction

VetAnimative is a modern, high-performance HTML template designed for Veterinary Clinics and Pet Shops. It features a unique Zero-Backend Admin Panel that allows you to manage content without needing a MySQL database.

2. Folder Structure

The zip package contains the following structure:

VetAnimative/
├── HTML/ -- All page files
├── assets/
│ ├── css/ -- Stylesheets
│ ├── js/ -- Main JavaScript
│ └── img/ -- Locally stored images
├── documentation/ -- This file
└── readme.txt

3. Admin Dashboard System

To manage your website content, you don't need PHP. Simply open admin.html in your browser.

Default Credentials:
Username: admin
Password: admin
  • Stats: View real-time income statistics.
  • Blog: Add, edit, or delete articles.
  • Team: Manage doctors and staff profiles.
  • Catalog: Manage products and services.
  • Gallery: Update your pet activity feed.

4. Customization

Changing Colors

Open assets/css/style.css and find the :root section to change the primary brand colors:

:root {
  --primary: #6c5ce7; /* Change your brand color here */
  --secondary: #00cec9;
}
