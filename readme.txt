THANK YOU FOR PURCHASING!
--------------------------------------------------------------------------------
VetAnimative is a modern, high-performance HTML5 template specifically 
designed for Veterinary Clinics, Pet Hospitals, and Pet Shops. It features
a unique Zero-Backend Admin System using LocalStorage technology.

MAIN FEATURES:
--------------------------------------------------------------------------------
- Full Content Management System (No Database/PHP required)
- Interactive Shopping Bag (Services & Products)
- Professional Admin Dashboard (Stats, Blog, Team, Catalog, Gallery)
- Fully Responsive (Mobile, Tablet, Desktop)
- Clean & Well Commented Code
- Custom 404 Error Page
- SEO Optimized with Alt Text tags

FOLDER STRUCTURE:
--------------------------------------------------------------------------------
- /HTML/           : Contains all the website files and assets.
- /Documentation/  : Contains the detailed user manual.
- readme.txt       : This file.

HOW TO START:
--------------------------------------------------------------------------------
1. Open /HTML/index.html in your browser to view the website.
2. To manage content, open /HTML/admin.html.
   - Default Username: admin
   - Default Password: admin
3. For customization, please refer to the /Documentation/ folder.
